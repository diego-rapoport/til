# Accent Colors

Use the `accent-color` property to style form controls with your branding color.

```html
<form>
    <input type="checkbox" id="test-check" checked />
    <label for="test-check">this is a checkbox</label>
</form>
```

```css
body {
  accent-color: #f5ad42;
}
```
